# Korean Ads Auditor

한국 광고주와 대행사를 위한 Codex Skill입니다.

Google/YouTube/Meta와 네이버, 카카오, 쿠팡, 당근 광고가 섞인 한국형 광고 계정을 CSV, 스크린샷, 붙여넣은 지표, 또는 사용자가 로그인한 광고 관리자 화면 기반으로 읽기 전용 진단합니다.

## Skill

스킬 본문은 [`korea-ads-auditor/SKILL.md`](korea-ads-auditor/SKILL.md)에 있습니다.

## What It Does

- Google Ads, YouTube Ads, Meta Ads 진단
- 네이버 검색광고/쇼핑검색광고/파워링크 진단
- 카카오모먼트/비즈보드/카카오톡 채널 광고 진단
- 쿠팡 상품광고, 당근 로컬 광고 진단
- 초보자용 로그인 링크 라우팅
- Codex 브라우저 기반 로그인 화면 읽기 전용 확인
- 표시광고, 후기/체험단, 개인정보/맞춤형 광고 리스크 점검

## Demo Prompt

```text
어디에 로그인해야 하는지 모르겠습니다.
제가 하는 광고 유형을 물어보고 맞는 로그인 링크를 골라주세요.
```

```text
처음 사용합니다. 네이버 검색광고를 진단하고 싶어요.
로그인 링크를 알려주고, 제가 로그인 완료라고 말하면 읽기 전용으로 화면을 확인해서 요약해 주세요.
```

## Safety

이 스킬은 기본적으로 읽기 전용입니다. 비밀번호, 2FA, 결제정보를 요청하거나 저장하지 않으며, 예산/입찰/캠페인 ON-OFF/저장/게시 버튼을 누르지 않도록 설계했습니다.
